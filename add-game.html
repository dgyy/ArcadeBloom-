🎮 Add New Games

Learn how to easily add new games to ArcadeBloom by editing just one file - games-data.js

Safe & Easy - No HTML Modification Required
1

Open games-data.js File

📁 Location: The games-data.js file is located in the root directory of your project, alongside index.html.

All game data is centralized in the games-data.js file. This approach ensures that:

  • Adding new games is safe and won't break existing pages
  • Game data is consistent across all pages (homepage, all-games, detail pages)
  • You don't need to modify multiple HTML files
  • Easy to maintain and organize game information
2

Add Your Game Data

Find the GAMES_DATABASE array in the file and add your new game object at the end: 

// Add your new game to the end of the GAMES_DATABASE array:
{
    id: 21, // 🔢 Use next available ID
    name: "Your Game Name", // 🎯 Display name
    slug: "your-game-name", // 🔗 URL-friendly version
    category: "puzzle", // 📂 Choose: action, puzzle, strategy, casual, sports, racing
    image: "https://example.com/game-image.png", // 🖼️ Game thumbnail (300x400 recommended)
    description: "Brief description of your game", // 📝 Short game description
    developer: "Your Name", // 👨‍💻 Developer/Creator name
    releaseDate: "2024", // 📅 Release date
    controls: "Arrow Keys", // 🎮 Control instructions
    rating: 4.5, // ⭐ Rating out of 5
    plays: 0, // 📊 Play count (starts at 0)
    featured: true, // ✨ Show on homepage? true/false
    tags: ["fun", "challenging", "new"], // 🏷️ Search tags
    gameUrl: "https://your-game-url.com", // 🌐 Actual game URL
    instructions: [ // 📋 How to play instructions
        "Step 1: How to start",
        "Step 2: Game controls", 
        "Step 3: How to win",
        "Step 4: Tips and tricks"
    ]
}

💡 Pro Tips:

  • • Make sure to increment the id number from the last game
  • • Set featured: true to show the game on the homepage
  • • Use descriptive tags to help users find your game
  • • Don't forget the comma after the closing brace },
3

Save & Test

✅ Save the File

Save the games-data.js file after adding your game data.

🔄 Refresh Pages

Refresh your browser to see the new game appear on all relevant pages automatically.
