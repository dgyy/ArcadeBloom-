# Cloud-only zero-cost growth automation

## Status
accepted

ArcadeBloom's unattended growth system will run without the owner's computer or new paid services: GitHub Actions hosts scheduled browser and build jobs, rate-limited GitHub Models supplies replaceable AI inference, and Cloudflare's free allowances host deployment and first-party measurement. Paid overage must remain disabled, providers must sit behind replaceable interfaces, and exhausted quotas or changed free policies pause the queue instead of creating charges or weakening review. This trades predictable throughput and availability for a fully online zero-cost bootstrap.
