# Grandfather existing indexed games during progressive review

## Status
accepted

The 2,019 games already published when automated evidence review was adopted may remain indexed provisionally, while every new game must pass the gate before indexing. Existing pages are reviewed progressively in search-value order and are improved, approved, or excluded when evidence warrants it; the exception never applies to later additions. This preserves launch speed and any emerging search signals at the cost of temporarily exposing pages that have not met the new standard.
